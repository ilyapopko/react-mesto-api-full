# Проект: "Место".
### Обзор
Практическая работа 15 для Яндекс.Практикума. Проект посвящен обмену фотографиями в интернете.
## Языки разработки
* HTML
* CSS
* JavaScript

## При разработке использованы:
* Нативная верстка
* Элементы flexbox
* Элементы grid
* Фреймворк React
* Фреймворк Express (Node)
* База данных - Mongo

## Модули:
* Валидация validator, joi/celebrate
* Шифрование - bcryptjs
* Логирование - winston
